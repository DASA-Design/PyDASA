# Rewrite Skill

Use this skill when asked to rewrite paragraphs or sections in the dissertation.

## Rules

1. Provide exactly 3 rewrite options unless told otherwise
2. Preserve the user's academic tone and writing voice
3. Place the edit in the exact location specified
4. Do not modify surrounding text
5. Do not add formatting symbols like ---
6. Use "we" and "practitioners" as pronouns
7. Use proper connector words between sentences
8. No em dashes — use commas, semicolons, or restructure
9. No colons as connectors between sentences
10. Keep prose concise — every sentence must carry information

## Process

1. Read the target section first
2. Understand the context (what comes before and after)
3. Write 3 options maintaining the user's voice
4. Wait for the user to choose
5. Apply the chosen option to the exact location
